Desktop Chess Game

Description

This is a simple desktop chess game developed in C# using WPF (XAML) for educational purposes.
The project was created as a solo assignment to practice object-oriented programming, user experience and design and basic game development.

The game supports two players playing against each other in person on the same computer.

Features

  Two-player local chess
  Standard chess rules
  WPF graphical user interface
  Windows-only application

Requirements

  Windows operating system
  .NET Framework / .NET Runtime

How to Run

  Download or clone the project.
  Open the solution in Visual Studio.
  Build and run the project.

Author

  Created as a solo educational project.
